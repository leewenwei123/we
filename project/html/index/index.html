Start Effect

遊玩須知！

每到農曆七月，恐怖密室逃脫成為搜尋熱點。

相比觀看電影或聽鬼故事，親身體驗恐怖場景更具刺激感。

配合陰森的背景音樂和突然出現的驚嚇元素，這樣的體驗常讓人難以入眠。

近年來，不僅密室逃脫，劇本殺也加入了許多恐怖元素。

每個人對恐怖的接受度不同，常見的恐怖元素有：

外觀驚悚：骷髏、假人屍體、血跡場景等，特別是突然移動的鬼怪。

昏暗光線：低光或無光場景讓人感到恐懼，特別是怕黑者。

感官剝奪：阻斷視覺或聽覺後，再給予刺激，例如進場前要求戴眼罩。

強制單獨行動：部分環節要求玩家獨自行動，對不喜歡落單的人是一種心理壓力。

突發驚嚇：利用 Jump Scare，在意想不到的時候驚嚇玩家。

每個人對恐怖的感受不同，難以預知遊戲中的驚嚇程度，就像半糖的珍奶對每個人的甜度感受也不同。
